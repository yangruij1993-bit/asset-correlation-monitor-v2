# 资产相关性与宏观配置：指标解读与计算逻辑 (Methodology)

本文档详细解释了 Asset Correlation Monitor 中使用的核心量化指标，以及在“Time Series (时间序列)”等页面中该如何解读这些数据以指导资产配置。

---

## 1. 什么是“滚动 (Rolling)”？

在传统的金融分析中，如果我们计算 VOO（标普500）和 TLT（长久期美债）在过去 10 年的相关性，只会得到一个单一的数字（比如 0.1）。但这掩盖了宏观周期的变化——在低通胀时期，股债通常是负相关的；但在高通胀时期，股债往往变成正相关（股债双杀）。

**“滚动窗口 (Rolling Window)”** 的计算方法是：
1. 设定一个固定的时间窗口长度，例如 `120 个交易日`（约 6 个月）。
2. 计算 [T-120天, T天] 这个窗口期内的相关性或波动率，记录为 T 天的数值。
3. 窗口向前滑动一天，计算 [T-119天, T+1天] 的数值，记录为 T+1 天的数值。
4. 将这些点连接成一条时间序列曲线。

通过“滚动”，我们能把静态的指标**动态化**，从而观察到市场结构的**拐点（Regime Shifts）**。

## 2. Time Series (时间序列) 页面指标解读

在 Dashboard 的第二页（Time Series），你可以选择查看不同指标的滚动变化：

### A. 滚动相关性 (Rolling Correlation)
*   **计算方法**：两只 ETF 在过去 N 天（默认 120 天）日收益率的 Pearson 相关系数，取值范围 `[-1, 1]`。
*   **红色的 0 轴虚线**：辅助线。高于 0 代表正相关（同涨同跌），低于 0 代表负相关（跷跷板效应）。
*   **如何解读**：
    *   **VOO vs TLT**：这是 60/40 组合的命门。如果线条在 0 以下，说明美债起到了很好的对冲作用（股市跌时债市涨）。如果线条向上突破 0 甚至达到 0.5 以上，说明股市和债市在同涨同跌（通常是因为市场主要矛盾从“经济衰退”变成了“通胀预期/利率预期”），此时债券失去了防御属性。
    *   **VOO vs GLD / PDBC**：观察商品和黄金在当前周期是否能提供股市之外的分散化收益。
    *   **VOO vs COWZ**：宽基与价值因子的相关性。如果相关性大幅下降，说明市场出现了强烈的风格分化（比如科技股崩盘，但现金牛企业抗跌）。

### B. 滚动波动率 (Rolling Volatility)
*   **计算方法**：该 ETF 在过去 N 天（默认 60 天）日收益率的标准差，并乘以 `sqrt(252)` 进行年化处理。
*   **如何解读**：
    *   衡量该资产在最近三个月的“情绪恐慌度”或“价格剧烈变动程度”。
    *   当 VOO 的滚动波动率急剧飙升时，往往对应着市场的快速回调或恐慌抛售。
    *   对于资产配置而言，当组合内大部分资产的滚动波动率都在上升时，通常意味着市场进入“高危期”，系统性风险增加，此时应考虑降减整体仓位或向短久期国债（如 VGSH）转移。

## 3. Signals (异常信号) 页面指标解读

仅仅看相关性曲线，我们有时很难判断“目前的 0.3 相关性到底是正常还是异常”。这就是 **Z-Score (标准分)** 和异常信号的作用。

### Z-Score 计算逻辑
对于任意一对资产（如 VOO-TLT）：
1.  提取它在全历史区间内的**滚动相关性曲线**。
2.  计算这条曲线的历史平均值 $\mu$ 和历史标准差 $\sigma$。
3.  提取**当前的最新相关性** $X$。
4.  **Z-Score** = $(X - \mu) / \sigma$

### 信号判定
*   **Normal (正常)**：$|Z| < 1.0$。当前两者的关系符合历史常态。
*   **Warning (警告)**：$1.0 \le |Z| < 1.5$。当前关系开始偏离历史中枢。
*   **Alert (异常/警报)**：$|Z| \ge 1.5$。这是极小概率事件。说明当前的宏观驱动力发生了根本性改变（例如，原本极度正相关的东西突然脱钩，或者原本负相关的对冲资产突然同向波动）。

### 配置启示 (Insights)
*   **如果对冲失效 (VOO-TLT 触发正向 Alert)**：缩短债券久期（把 TLT 换成 VGSH），或者增加大宗商品（PDBC）/黄金（GLD）来代替长债做防御。
*   **如果风格脱钩 (VOO-COWZ 触发负向 Alert)**：说明大盘指数与高自由现金流企业走势严重背离。这通常是“抱团瓦解”的前兆或发生中，超配 COWZ 可以提供纯粹的 Alpha（超额收益）。
